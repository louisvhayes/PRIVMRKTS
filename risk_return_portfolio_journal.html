01

Returns: Dollar, Percentage & Holding Period

The starting point for all investment analysis is measuring return precisely. A return has two components — income (dividends, coupons) and capital appreciation (price change). The total return captures both. Understanding exactly how returns are calculated, and particularly how they compound over time via the holding period return, is the foundation for everything that follows in portfolio theory.

Dollar Return
Dollar Return = P_{t+1} + D_{t+1} − P_t
P_t = price at start of period · P_{t+1} = price at end · D_{t+1} = dividend paid
Raw profit/loss in currency units — not scaled to investment size
Percentage Return
r_{t+1} = (P_{t+1} + D_{t+1} − P_t) / P_t
= Dividend yield + Capital gains yield
Scaled to investment size — allows comparison across different investments
Holding Period Return (n years)
HPR = (1+r₁)(1+r₂)···(1+rₙ) − 1
Total return from holding investment for n years, assuming reinvestment of dividends
Captures the compounding effect — returns multiply, they don't add
Walmart example — full calculation
Shares purchased100 @ $25
Initial investment P₀$2,500
Dividends received D₁$20
Ending price P₁$30 × 100 = $3,000
Dollar return$3,000+$20−$2,500 = $520
% return$520/$2,500 = 20.8%
Decomposition: dividend yield = $20/$2,500 = 0.8%; capital gain = ($3,000−$2,500)/$2,500 = 20%. Together: 0.8% + 20% = 20.8% ✓
HPR example — 4 years
Annual returns: r₁=10%, r₂=−5%, r₃=20%, r₄=15%
HPR = (1.10)(0.95)(1.20)(1.15) − 1
= 1.4421 − 1 = 44.21% total return over 4 years
Note: arithmetic sum = 10−5+20+15 = 40%. But compounding gives 44.21% because positive returns compound on previously accumulated gains. The order of returns doesn't affect the HPR — only their values matter.
Wealth growth — HPR compounding vs simple summation ($100 initial investment)
Returns: +10%, −5%, +20%, +15%
Compounded wealth (HPR) Simple sum (not compounded)
02

Arithmetic vs Geometric Average Returns

Two different averages answer two different questions. The arithmetic average is the best estimate of what you expect to earn next period. The geometric average tells you your actual annual compound return over a past period — it measures wealth growth. They only agree when returns are constant. In the presence of volatility, the geometric average is always lower — a phenomenon called volatility drag.

Arithmetic Average Return
r̄_A = (1/n) × Σ rᵢ
Simple mean of all period returns
Best estimate of expected return in a single future period
Does NOT tell you how much wealth grew over the full period
Geometric Average Return
r̄_G = [Π(1+rᵢ)]^(1/n) − 1
Constant annual growth rate that produces the same final wealth
Equivalent to annualised HPR
Always ≤ arithmetic average (equality only when returns are constant)
Volatility drag — why r̄_G < r̄_A
Classic example: r₁=+50%, r₂=−50%, W₀=$100
Arithmetic average(50%−50%)/2 = 0%
W₁ = 100 × 1.50$150
W₂ = 150 × 0.50$75
Geometric average√0.75 − 1 = −13.4%
Despite an arithmetic average of 0%, you lost 25% of your wealth. The +50% gain applies to a $100 base; the −50% loss applies to a $150 base. Compounding is multiplicative — losses bite harder than gains help when applied to different base amounts.
Approx relationship: r̄_G ≈ r̄_A − σ²/2 (volatility drag)
Volatility drag — geometric vs arithmetic average (σ impact)
r̄_A = 10% fixed · varying σ
As volatility σ increases, geometric average falls further below arithmetic average. The gap is approximately σ²/2 — higher risk means lower compound wealth growth for the same average return.
Which average to use — decision rule
  • Arithmetic average — use when estimating the expected return for a single future period (e.g., what return do I expect next year?)
  • Geometric average — use when measuring past wealth growth, or estimating the compound annual return over a multi-year horizon
  • For long-term financial planning (e.g., retirement savings), the geometric average is more relevant because it tells you the actual annual growth rate of your wealth
  • For option pricing and single-period models (e.g., CAPM), use the arithmetic average

Historical Returns, Risk & the Risk Premium

A century of US market data reveals a consistent and powerful pattern: higher returns come with higher risk. This is the empirical foundation of the risk-return trade-off — not just a theoretical prediction, but a fact observed across every major asset class over many decades. The data also quantifies the equity risk premium — the extra return investors have historically earned for bearing stock market risk.

Portfolio Theory — Expected Returns & Weights

Portfolio theory begins with a simple but powerful insight: the expected return of a portfolio is just the weighted average of the expected returns of its components. Weights must sum to one. But the variance — the measure of risk — is not a simple weighted average. It depends critically on how the assets move together, which is measured by covariance. This distinction is the entire basis of diversification theory.

Portfolio Weight
xᵢ = (Value of investment i) / (Total portfolio value)
Weights must sum to 1: Σxᵢ = 1
Negative weights are possible (short positions)
Portfolio Return
R_p = x₁R₁ + x₂R₂ + ··· + xₙRₙ = Σxᵢ Rᵢ
Realized return is weighted average of component returns
Weights at purchase, not realised weights — see Disney/Coke example
Expected Portfolio Return
E[R_p] = Σ xᵢ E[Rᵢ]
Expected return = weighted average of expected returns
The weighting of expected returns is always linear — unlike variance
Example 1 — Disney & Coca-Cola portfolio
200 Disney @ $30 ($6,000) + 100 Coke @ $40 ($4,000) = $10,000
Initial Disney weight x_D6,000/10,000 = 60%
Initial Coke weight x_C4,000/10,000 = 40%
Disney return R_D36/30 − 1 = +20%
Coke return R_C38/40 − 1 = −5%
R_p = 0.6×20% + 0.4×(−5%) = 10% ✓
New portfolio value: 200×$36 + 100×$38 = $11,000 (gain of $1,000 = 10% ✓). After the price change, weights shift: Disney = $7,200/$11,000 = 65.45%. Without rebalancing, weights drift toward the outperforming stock.
Example 2 — Ford & Tyco expected return
$10k in Ford (E[R]=10%), $30k in Tyco (E[R]=16%)
x_Ford = 10k/40k0.25
x_Tyco = 30k/40k0.75
E[R_p] = 0.25×10% + 0.75×16% = 14.5%
The portfolio expected return (14.5%) lies between the two individual returns (10% and 16%), weighted by the capital allocation. More capital in higher-return Tyco pulls the portfolio return above the midpoint.
05

Covariance, Correlation & Portfolio Variance

The risk of a portfolio depends not only on the individual risks of its components but critically on how those components move together. Covariance measures the direction and magnitude of co-movement; correlation normalises this into a dimensionless measure between −1 and +1. The portfolio variance formula makes clear that combining assets with low or negative correlation reduces risk — this is the mathematical foundation of diversification.

Covariance
Cov(Rᵢ,Rⱼ) = E[(Rᵢ−E[Rᵢ])(Rⱼ−E[Rⱼ])]
Historical estimate: (1/(T−1)) × Σ(Rᵢ,t−R̄ᵢ)(Rⱼ,t−R̄ⱼ)
Positive = tend to move together · Negative = tend to move oppositely
Correlation
Corr(Rᵢ,Rⱼ) = Cov(Rᵢ,Rⱼ) / [SD(Rᵢ) × SD(Rⱼ)]
Always between −1 and +1
−1 = perfect negative correlation (moves exactly opposite)
+1 = perfect positive correlation (moves exactly together)
0 = uncorrelated (no linear relationship)
Two-Stock Portfolio Variance
Var(R_p) = x₁²σ₁² + x₂²σ₂² + 2x₁x₂Cov(R₁,R₂)
Or equivalently: x₁²σ₁² + x₂²σ₂² + 2x₁x₂ρ₁₂σ₁σ₂
SD(R_p) = √Var(R_p) — always take the square root for volatility
North Air, West Air, Tex Oil — diversification in action
All three stocks: avg return = 10%, volatility = 13.4%

The Dell/Alaska portfolio (34.2%) is less volatile than either stock alone (Dell=50%, Alaska=38%). Lower correlation (0.19 vs 0.62) creates much more diversification — even though Alaska Air is individually riskier than Microsoft, its low correlation with Dell makes it a better diversifier.
Key lesson: a riskier asset with lower correlation can reduce portfolio risk more than a safer asset with high correlation

The Efficient Frontier & Portfolio Optimisation

With two or more assets, there are infinitely many possible portfolios. The efficient frontier identifies the optimal subset: the portfolios that provide the highest possible expected return for each level of risk (or equivalently, the lowest risk for each level of expected return). Rational investors should only hold portfolios on the efficient frontier — any other portfolio is suboptimal.

Consider a portfolio combining Intel (26% expected return, 50% volatility) and Coca-Cola (6% expected return, 25% volatility). By varying the weight in Intel from 0% to 100%, we trace out a curve in risk-return space. Some portfolios on this curve are efficient — no other portfolio has both higher return and lower risk. Others are inefficient — dominated by a better alternative.

Long & Short Positions — Extending the Frontier

A long position is simply owning a security — you profit if the price rises. A short position involves borrowing a security, selling it now, and buying it back later — you profit if the price falls. Short selling allows portfolio weights to be negative, extending the investment universe beyond the constrained frontier. This expands the set of achievable risk-return combinations, allowing both higher expected returns and new hedging strategies.

Long vs short — mechanics
Long payoffP_T − P_0 (profit if price rises)
Short payoffP_0 − P_T (profit if price falls)
Short weight xᵢNegative
Weights still sum to1.0 (100%)
When you short $10k of Coke and invest $30k in Intel (using the $10k short proceeds + $20k cash), you hold:
x_Intel = $30k/$20k = 150%
x_Coke = −$10k/$20k = −50%
Check: 150% + (−50%) = 100% ✓
Short sale example — Intel long / Coke short
Cash available$20,000
Short $10k Coke → invest in Intel$30,000 in Intel
x_Intel30,000/20,000 = 150%
x_Coke−10,000/20,000 = −50%
E[R_p] = 1.50×26% + (−0.50)×6% = 36%
SD(R_p) = √(1.5²×0.50² + 0.5²×0.25²) = 76.0%
Assumes ρ = 0 (Cov = 0)
Short selling amplifies both return (36% vs 26% for Intel alone) and risk (76% vs 50%). It moves the portfolio beyond the individual stocks — above and to the right of the constrained frontier.
Short selling is a leverage strategy — magnifies both gains and losses

All formulas — exam reference
  • r = (P₁ + D₁ − P₀) / P₀ — total percentage return = dividend yield + capital gain
  • HPR = Π(1+rᵢ) − 1 — holding period return; compounding over n periods
  • r̄_A = (1/n) Σrᵢ — arithmetic average; best single-period expected return estimate
  • r̄_G = [Π(1+rᵢ)]^(1/n) − 1 — geometric average; measures wealth growth; ≤ r̄_A
  • xᵢ = Value_i / Total portfolio — portfolio weight; Σxᵢ = 1 (negative = short position)
  • R_p = Σxᵢ Rᵢ — portfolio return is weighted average of component returns
  • E[R_p] = Σxᵢ E[Rᵢ] — expected portfolio return is linear in weights
  • Cov(i,j) = (1/(T−1)) Σ(Rᵢ−R̄ᵢ)(Rⱼ−R̄ⱼ) — covariance from historical data
  • Corr(i,j) = Cov(i,j) / [SD(i)×SD(j)] — correlation; scale-free, always in [−1,+1]
  • Var(R_p) = x₁²σ₁² + x₂²σ₂² + 2x₁x₂ρσ₁σ₂ — two-stock portfolio variance
  • Cov(i,j) = Corr(i,j) × SD(i) × SD(j) — recover covariance from correlation table
  • Risk premium = E[R_asset] − R_f — excess return above the risk-free rate
Common mistakes & misconceptions
  • Geometric avg ≠ arithmetic avg — even if arithmetic avg = 0%, you can still lose money (50%, −50% example: final wealth = 75%)
  • Portfolio variance is NOT a weighted average — unlike expected return, variance depends on covariances; you cannot just weight individual variances
  • Lower individual risk ≠ better diversifier — a riskier stock with lower correlation can reduce portfolio risk more than a safer stock with high correlation
  • Covariance and correlation are different — covariance has units (percentage²); correlation is dimensionless [−1,+1]. Use correlation to compare across pairs.
  • Short sale weights are negative but still sum to 1 — e.g., 150% + (−50%) = 100%. Forget this and your portfolio return formula will be wrong.
  • Higher risk premium ≠ "invest only in stocks" — the premium exists because stocks are risky; bonds still serve important roles for risk-averse investors and those with short horizons
  • HPR order of returns doesn't matter — +50%,−50% gives the same final wealth as −50%,+50%. What matters is the set of returns, not their sequence.
  • 100% Coke is inefficient — a portfolio of 40% Intel / 60% Coke has the same volatility (25%) but 8% higher expected return. Refusing to diversify has a real cost.
STEP 1 — Compute returns correctly
r = (P₁+D₁−P₀)/P₀ · HPR = Π(1+rᵢ)−1
Always include both price appreciation and dividend income. Use HPR for multi-period returns. Remember: compounding is multiplicative, not additive. A 50% gain followed by a 50% loss leaves you with only 75% of your initial wealth, not 100%.
STEP 2 — Understand arithmetic vs geometric average
r̄_G ≤ r̄_A — volatility drag is real and costs compound returns
Use arithmetic average for single-period expected returns (CAPM inputs). Use geometric average for measuring past wealth growth or projecting long-term compound returns. The gap between them is approximately σ²/2 — higher volatility means more drag.
STEP 3 — Interpret historical data: risk-return trade-off is real
Higher return requires bearing higher risk — quantified by σ
The US data (1926–2005) confirms the trade-off empirically. Large stocks: 12.3% avg, 20.2% σ. T-bills: 3.8% avg, 3.1% σ. The equity premium of 8.5% compensates investors for the higher volatility — it is not "free money" but compensation for risk.
STEP 4 — Build portfolios; expected return is linear but variance is not
Var(R_p) = x₁²σ₁² + x₂²σ₂² + 2x₁x₂Cov(1,2)
Expected return is simply a weighted average — easy. Variance involves the covariance term — this is where diversification lives. A negative or low covariance creates the diversification benefit. Compute covariance from correlation: Cov = ρ×σ₁×σ₂.
STEP 5 — Identify efficient portfolios; reject the inefficient ones
Efficient: no better portfolio exists with same risk or same return
Plot all portfolios in risk-return space. The efficient frontier is the upper-left boundary. Any portfolio below this line is dominated. Short selling extends the frontier but also increases both return and risk. Rational investors hold only efficient portfolios.
One-line synthesis — Risk, Return & Portfolio Theory
The risk-return trade-off is an empirical law: assets that have delivered higher long-run returns have always done so by bearing more risk. But portfolio theory reveals that risk is not fixed — by combining assets with low or negative correlations, an investor can reduce risk without sacrificing return. The efficient frontier is the mathematical expression of this insight: it is the set of all portfolios that a rational, risk-averse investor should even consider holding.
